# 🧠 Customer Churn Prediction using ANN

## 🚀 Overview
This project predicts whether a customer will leave a bank using an Artificial Neural Network.

---

## 📊 Dataset
- Churn Modelling dataset  
- Features: Age, Balance, Geography, etc.  
- Target: Exited (0 or 1)

---

## ⚙️ Approach
- Label Encoding + One Hot Encoding  
- Train-test split  
- Feature scaling  
- Built ANN with:
  - 2 hidden layers (ReLU)
  - Output layer (Sigmoid)

---

## 🧠 Model Architecture
- Dense (6 units, ReLU)
- Dense (6 units, ReLU)
- Dense (1 unit, Sigmoid)

---

## 📈 Results
- Accuracy ~85%+ (approx)

---

## 🧠 Insights
- ANN captures non-linear patterns better than traditional ML  
- Feature scaling is critical  
- Sigmoid used for binary classification  

---

## 🏆 Conclusion
ANN provides strong performance for customer churn prediction.
